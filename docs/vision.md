# NoBavel Vision

NoBavel is a GIS-style construction coordination platform that synchronizes revisions, layers, and workflows across all construction disciplines.

The platform creates a single source of truth around a live spatial building model.

NoBavel is NOT CAD software.

Core goals:
- eliminate construction coordination chaos
- synchronize revisions across teams
- provide realtime visibility
- improve accountability
- reduce expensive field mistakes
- spatially organize all project data

Primary users:
- architects
- structural engineers
- electricians
- plumbers
- HVAC contractors
- glazing contractors
- paving contractors
- flooring contractors
- inspectors
- project managers
- developers

Core MVP:
1. spatial plan viewer
2. hierarchical layer system
3. revision tracking
4. comments/annotations
5. notifications
6. approvals

Non-goals:
- replacing AutoCAD
- full CAD editing
- BIM authoring
